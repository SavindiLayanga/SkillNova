# SkillNova
AI-powered platform that analyzes user skills, identifies skill gaps, and provides personalized career recommendations using NLP and intelligent job matching.

## Project Structure
This project is separated into two parts:
- `frontend`: React/Vite web application
- `backend`: Node/Express backend server

## How to Run

### Run Everything Concurrently (from the root directory):
```bash
npm install
npm run dev:all
```

### Run Separately:

**1. Frontend:**
```bash
cd frontend
npm install
npm run dev
```
The frontend will be available at http://localhost:5173

**2. Backend:**
```bash
cd backend
npm install
npm run server
```
The backend server will run on http://localhost:5000
